1.环境

注意需要导入spring-websocket和websocket-api包。其余架包正常ssm即可，可自行百度

2.后台代码

  1. @ServerEndpoint：把当前类变成websocket服务类
  2. @OnOpen： 连接时执行
  3. @OnClose：关闭时执行
  4. @OnMessage：收到消息时执行
